🚀 Getting Started
Prerequisites
Before you install HORRAH, make sure you have the following set up:

Project Hyacinth up and running locally or on your server.

PHP (version X.X or higher)

Composer (if managing dependencies)

Installation
Clone the repository:

Bash
git clone https://github.com/yourusername/HORRAH.git
Navigate to the directory:

Bash
cd HORRAH
Install dependencies:

Bash
composer install
Configuration:

Copy .env.example to .env.

Update the .env variables, ensuring the HYACINTH_API_KEY or connection URLs point to your active Project Hyacinth instance.

🔗 Integration with Project Hyacinth
To get HORRAH communicating with Hyacinth:

Ensure both applications are running.

In Project Hyacinth's configuration, register HORRAH's local URL/IP.

Verify the connection by hitting the /health endpoint on HORRAH from the Hyacinth dashboard.
